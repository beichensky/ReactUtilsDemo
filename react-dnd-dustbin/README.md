## react-dnd-dustbin 项目内容
该项目是一个 `Dustbin` 类型的 `Demo` ，里面是 `react-dnd` 的一些基本用法。代码量不多，通俗易懂。可以将项目拉下来看看运行效果，对照代码查看实现逻辑，效果更佳。

## 项目运行

### 准备工作
项目下载完成之后，打开项目，打开命令行，运行以下命令:
``` bash
# 打开项目目录
$ cd react-dnd-dustbin

# 安装项目所需插件
$ npm install
```



### 查看运行结果
在命令行运行：
``` bash
$ npm run start
```
即可在浏览器中看到项目运行效果，对照代码查看对应逻辑。
